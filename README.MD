# Dogs REST API
Value of  JDBC route from src/main/resources/application.properties:
```spring.datasource.url=jdbc:h2:mem:dogdata```
H2 console enabled with:
```spring.h2.console.enabled=true```
Path to h2:
```spring.h2.console.path=/h2```
